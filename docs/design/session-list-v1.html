デザイン上の判断メモ

迷った点・意図的な変更点：
  • 録音長・日時は SwiftUI 実装（SessionRowView）では末尾配置だが、本モックでは行内の情報密度を優先し、タイトル直下のメタ行にまとめて配置。時刻表記の位置自体は仕様上の制約がないため据え置きで問題ないと判断。
  • 録音モードアイコン（🎙 Mic / 🖥 System / 🎙🖥 Both）は絵文字のみで自己完結させ、外部アイコンフォント依存を避けた。実装では SF Symbols（mic / display / rectangle.2.swap 等）に置き換わる想定。
  • 話者チップは3話者までを A=青（--spk-a）/ B=緑（--spk-b）/ C=紫（--spk-c）で固定し、v2 の話者色ポリシーをそのまま踏襲。4人以上のスケーリングは v2 同様オープンクエスチョンのまま。
  • 空状態は実装中の ContentUnavailableView("録音がありません", systemImage: "waveform", description: "新しい録音を開始してください") の文言・情報量に忠実。CTA ボタンは追加していない（現行実装に合わせてスコープ外とした）。
  • 新規 CSS カスタムプロパティは追加していない。行ホバーは color-mix(in srgb, var(--text) 5%, transparent) で既存トークンから合成。
