sokki — SettingsView

設定画面のモックです。RecordingView v2 のトークンを流用し、現行 SettingsView.swift の3タブ（文字起こし・話者分離・LLM）に加えて、並行実装中の TASK-9.2（外観の自動追従＋手動切替）に対応する「外観」タブを追加しています。「外観」の システム / ライト / ダーク を切り替えると、実際にこのモックのテーマが変わります。

バージョン: v1 対象画面: SettingsView（設定） フェーズ: Phase 1 MVP — backlog TASK-9.3 / Issue #60 プラットフォーム: macOS 15+ / Apple Silicon

UI モック — SettingsView（インタラクティブ）

デザイン上の判断メモ

迷った点・意図的な変更点：
  • ウィンドウ枠への統合：現行実装は SettingsView を独立した 480×320 の TabView ウィンドウとして表示するが、本タスクの要件（「ウィンドウ枠・サイドバー構成は session-list-v1 と同一」）に従い、他画面と同じサイドバー付きシェル内のコンテンツとして再構成した。実装が別ウィンドウのままか、他画面と統一されたペインになるかは TASK-9.2 側の設計判断に委ねる。
  • 「外観」タブの追加と並び順：要件により新規追加。macOS 標準の設定画面の慣習（外観・一般系の設定を先頭に置く）に倣い、タブの先頭に配置した。現行実装の3タブ（文字起こし・話者分離・LLM）は既存の並びのまま維持している。
  • 「外観」ピッカーを実際に機能させた：システム/ライト/ダークを切り替えると、このモック自体のテーマ（.dir-manuscript / .dir-console）が実際に変わる。「システム」を選んだ場合は、ページ上部の「テーマ」トグル（システムの現在の外観設定を模したもの）に追従する。これにより preferredColorScheme + システム追従の実際の挙動を体験できるようにした。
  • タブアイコン：現行 SwiftUI 実装の Label に使われている SF Symbols（waveform / person.2 / brain）に対応する絵文字（〰️ / 👥 / 🧠）を採用し、実装との対応関係が分かるようにした。「外観」用に🌗を追加。
  • 話者ラベルのロケール切替トグルは省略：他3画面と異なり、本画面には話者名や話者ラベル文字列が一切表示されない（「話者数」はあくまで人数の数値設定）ため、ja/en 切替トグルは設置していない。
  • トグルスイッチの ON/OFF 双方向対応：RecordingView v2 の .toggle/.sw は「常時 ON（翻訳オン）」の1状態のみを表現していたが、本画面の「話者分離を有効にする」は実際に ON/OFF を行き来する必要があるため .sw.off 修飾クラスを追加した。新規カスタムプロパティではなく、既存の --line（OFF背景）をそのまま使っている。
  • Stepper・Slider の値表示：現行実装の文言「話者数: 自動 / N人」「%.2f 表示の照合閾値」をそのまま踏襲し、クリック・ドラッグで実際に値が更新されるようにした（0〜10人・0.60〜0.95の範囲も実装に合わせた）。
  • 将来項目（Phase 2.5 の翻訳設定等）は本モックの対象外とし、含めていない。
